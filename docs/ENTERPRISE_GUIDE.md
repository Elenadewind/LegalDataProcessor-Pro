## Руководство для корпоративных клиентов
### 1. Интеграция с ERP
- **SAP**: используйте `sap_connector.py`.  
- **Oracle**: см. `oracle_integration.md`.  
- **1С**: API‑шлюз в `api/1c_gateway.py`.  
### 2. Развёртывание
- **Локально**:  
  ```bash
  docker-compose up -d
 В облаке: образы для AWS/GCP в deploy/.
3. Безопасность
Данные не передаются третьим лицам.
Шифрование AES‑256.
Аудит логов: logs/audit.log.
  ### 4. Мониторинг

В текущей версии:
- Основные логи записываются в `logs/app.log` (формат: JSON).
- Дашборд и система алертов не включены.

В планах:
- Веб‑дашборд с метриками (версия 1.2).
- Настройка алертов через `config/alerts.yaml` (версия 1.3).
